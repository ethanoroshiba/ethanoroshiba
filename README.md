# About Me:
Hey — I’m Ethan!  <br>I’m a software engineer at [Astria Labs](https://www.linkedin.com/company/settler-labs/posts/?feedView=all) working across distributed systems, gRPC APIs, and AI-integrated backend infrastructure.  <br><br>My day-to-day involves designing reliable interfaces and event-driven systems in **Rust**, **Go**, and **Python** — from sequencer APIs and mempool caches to LLM-powered job orchestration and test automation.  <br><br>Recent work includes:<br>- Designing and shipping type-safe gRPC APIs serving internal and external clients  <br>- Building Faktory + Redis job systems with NATS event streams (~16 k events/s)  <br>- Creating AI-driven mock frameworks for streaming LLM responses, cutting CI cost and runtime  <br>- Contributing to the design and implementation of core protocol features powering the Astria shared sequencer<br><br>I love solving reliability problems, connecting complex systems, and designing reliable software around correctness and maintainability.
